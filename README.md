# Student-Marks-Management-System
Overview

The Student Marks Management System is a console-based application developed using Python to simplify the management of student academic records. The application allows users to add, view, update, and manage student marks efficiently while providing hands-on experience with Python programming concepts and data handling.

Features
Add new student records
View stored student information
Update marks and details
Manage multiple student records
Simple menu-driven interface
Easy-to-understand console application

Technologies Used
Python

How It Works
The application provides a menu-driven interface.
Users can enter student details and marks.
Student information is stored and managed within the program.
Records can be viewed and updated whenever required.

Python Programming
